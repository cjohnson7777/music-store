# Music Store E-commerce Website
This project is a static e-commerce storefront designed to practice frontend technologies, focusing on excellent User Experience (UX), responsive design, and integration with third-party APIs.

# Key Features:
- Optimized UX: Smooth animations and clean, intuitive navigation using CSS and JavaScript to create an engaging browsing experience.
- Responsive Design: CSS Media Queries and flexible layouts to ensure the store renders perfectly and is accessible on all devices.
- Explore New Song Lyrics: Integrated a third-party API to fetch and display unique, randomized content.

# Tech Stack:
- HTML
- CSS
- JavaScript

# Setup and Run Locally:
1. Clone this repository
   ```
   git clone https://github.com/cjohnson7777/music-store/tree/main
   ```
2. Launch by opening the main 'index.html' file in your web browser to view the site.
